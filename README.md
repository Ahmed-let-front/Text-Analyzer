## 🚀 Engineering Highlights

- **Logical Properties:** Fully implemented using `inline-size`, `margin-block`, and `inset-inline-start`, ensuring the layout is globally compatible (RTL/LTR ready).
- **Modern CSS Architecture:** Uses **CSS Nesting** (strict 3-level depth) for maximum maintainability and clean selector specificity.
- **Scroll-Driven Animations:** Leverages cutting-edge `animation-timeline: view()` and `scroll()` for a fluid UI without the overhead of heavy JavaScript libraries.
- **V8 Memory Awareness:** Built with an understanding of JS engine internals—treating strings with the performance respect they deserve.
